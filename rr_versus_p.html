Round Robin vs Priority Scheduling

Project Objective: Implement and compare Round Robin and Priority Scheduling using the same workload, then analyze fairness, urgency handling, response time, waiting time, turnaround time, and starvation risk.

Priority Rule Used: Lower priority number means higher priority. Priority Scheduling here is preemptive. Tie-breaker: earlier arrival time, then smaller input order.
