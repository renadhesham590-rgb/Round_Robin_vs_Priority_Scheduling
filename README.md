# Round_Robin_vs_Priority_Scheduling
C3: OS algorithm comparison/ scheduling project

Project Description
This project compares 2 CPU scheduling algorithms: Round Robin and Priority Scheduling(Pre-emptive). Both algorithms use the same process table to analyze fairness, urgency, WT(waiting time), TAT(turnaround time), RT(response time), and starvation risk.
